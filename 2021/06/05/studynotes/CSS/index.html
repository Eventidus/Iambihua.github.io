CSS介绍

Cascading Style Sheets层叠样式表的简称。

用来表现HTML或XHTML。

用处

  • 外部样式表

    • 适用于多个页面应用同一样式的场景
    • 在HTML的头部head标签中引入

  • 内部样式表　

    • 适用于单个页面应用某个样式的场景
    • 在HTML头部标签的style标签中引入

  • 内联样式

    • 适用于某个特定HTML元素设置样式的场景
    • 在HTML标签中使用style属性

优点

  • 内容和表现的分离。
  • 减少代码量，提高浏览器的浏览速度。

层叠次序（优先级由上到下增高）

  • 浏览器缺省设置
  • 外部样式表
  • 内部样式表
  • 内联样式

CSS规则

  • 选择器：通常是需要设置样式的HTML元素

  • 注意

    • 一个属性和其对应的属性值组成一条声明，属性和属性值由冒号分隔。
    • 声明组用大括号围起来，以分号结束。

  • 声明

    • 属性
    • 属性值

颜色值

  • 十六进制数表示

  • RGB值

    • 0-255之间的值
    • 百分制，0也要有百分号

  • 颜色名称

CSS ID和Class 选择器

ID

  • ID选择器以#号开头
  • ID选择器在HTML表现为ID属性
  • 主要为特定HTML元素设置特定样式

Class

  • Class选择器以　．开头
  • Class选择器在HTML表现为Class属性
  • 即可以为特定HTML元素设置样式，也可为多个元素设置样式
