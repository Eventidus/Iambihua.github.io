this 关键字

  1. this 总是返回一个对象。

  2. this 的值为属性或方法当前所在的对象(环境)。

  3. 使用场合 1. 在全局环境下，this 的值为顶层环境 window。 2. 在构造函数中，this 指向实例对象。 3. 在对象的方法中，this 指向当前方法所在的对象，但如果对一个对象进行赋值时，this 的指向就会改变了。

  4. 多层 this 的情况下，可以考虑使用一个变量指向外层 this。

  5. 回调函数中的 this 指向当前环境中的全局对象。

  6. Function.prototype.call()

    1. 用来指定 this 的指向。
    2. call 方法的参数如果为空,null,或者 underfined，则默认指向 window 对象。
    3. 如果只有一个参数，并且参数为对象，则 this 指向这个参数对象。
    4. 如果只有一个参数，并且参数为原始值，则 this 指向这个原始值的包装对象。
    5. 如果有多个参数，则第一个参数为 this 指向的对象，后面的参数为函数调用时需要的参数。

  7. Function.prototype.apply()

    1. 跟 call 方法类似，也是用来指定 this 的指向。
    2. 如果只有一个参数，并且参数为对象，则 this 指向这个参数对象。
    3. 如果有多个参数，则第一个参数为 this 要指向的对象，第二个参数为参数数组。

  8. Function.prototype.bind()

    1. bind 方法用于将函数中的 this 绑定到某个对象，并且返回一个新函数。
    2. bind 方法也可以用于绑定参数。
    3. 如果只有一个参数，并且参数为对象，则 this 指向这个参数对象。
    4. 如果有多个参数，则第一个参数为 this 指向的对象，后面的参数为函数调用时需要的参数。
    5. 注意事项：
      1. 每次调用会返回一个全新的函数
