VueRouter

  1. 导入 vue 及 vue-router 库

  2. 安装 vue-router 插件

  3. 导入路由组件

  4. 创建路由数组即 routes 数组，数组一般有 1. path 属性定义访问路径 2. component 属性指定组件名 3. redirect 属性用于路由重定向 4. meta 属性是一个对象，为路由记录添加其他额外信息，通过这个信息以实现控制路由的访问等功能

  5. 创建路由器对象即通过 new VueRouter()构造函数,将 routes 数组传入构造函数参数里,其他参数：mode:指定 url 地址是否带有 # 号

  6. 导出路由配置文件

  7. 在 main.js 主入口文件中引入路由规则文件，并且注册路由

  8. vue 路由是一种键值对，它表示的是路径与路由组件之间的映射

  9. 路由的$route 属性用于获取路由相关信息如 query,params，fullPath(可用于组件的隐藏或显示)等

    1. query 查询参数，也就是 url 地址?后面的参数,不需要预先占位。
    2. params 路径参数相当于 url 地址的一部分，用于动态指定 url 地址，需要预先占位。

  10. 路由传参的时候，如何指定 params 参数可传也可不传

    1. 在配置路由规则的时候，在路由占位的后面加上一个问号。
    2. 如果同时提供了 path 和 params，那么 params 会被省略，但是 query 不会。
    3. 如果同时提供了 name 和 params，那么 params 可以正常工作。

  11. 路由的$router 属性用于编程式导航进行路由跳转(Push|replace)等

  12. 路由有声明式导航(router-link)和编程式导航，其中声明式导航必须有 to 属性

  13. 三种路由跳转方法

    1. push()方法会在路由历史栈中添加一条新的记录
    2. replace()方法只是替换，而不会向历史栈中添加一条新的记录
    3. go()表示在历史栈中前进或后退多步

  14. 路由的前进或者后退

    1. router.forward()前进一步
    2. router.back()后退一步

  15. 三种使用 props 传参的方式

    1. 布尔模式：props=true,这时 props 的值为 params
    2. 对象模式：props 的值为一个对象
    3. 函数模式：返回一个 props 对，route => ({ })
