Hexo 版本升级指南

1.更新升级 hexo

npm install hexo-cli -g

2.查看可更新 npm 依赖包并升级

安装 npm-check 包

npm install npm-check -g

检查可更新依赖包并升级

npm-check -u

or

安装 npm-upgrade 包

npm install -g npm-upgrade

检查可更新依赖包件并升级

npm-upgrade

4.更新

npm update -g
npm update --save
