Markdown 介绍

什么是 Markdown?它是一种轻量级的文本标记语言，使用它可以更方便地编辑文档。

很多网站都利用它来撰写文档或用于在论坛发消息等等。

说到这大家可能就像在听天书一样，因为搞不懂专业用语。

事实上这并不是什么计算机编程语言，而是通用的普通人都可以学的标记语言，贱简单呐。

在讲解常见的 Markdown 语法之前，我先向大家介绍一下富文本编辑器。

富文本编辑器其实很常见的，比如 Word，WPS 的编辑页面就是一种富文本文本编辑器。

那么它与支持 Markdown 语法的编辑器有何不同？

最大的区别在于：富文本编辑器对文本内容的编辑，大多是通过键盘快捷键或者点击编辑按钮比如加粗，斜体，标题设置等来编辑的。

Markdown 编辑器就不一样了，在这里举个例子，如果要设置一级标题，应该怎么处理呢？

很简单 ，在开头打一个#号就可以了，比如 # 这是一级标题(注意#号留有一个空格)，那么二级标题呢？so easy，在开头打两个#号就可以了。

是不是很简单？所以学习 Markdown 语法，并不需要太费脑费劲，只需要记住大部分常用的语法格式即可。
下面是我总结的使用 Markdown 语法编写文章的好处：

  • 沉浸式写作，完全不会太分心，不用再点击鼠标或者敲键盘编辑文章了。
  • 方便快捷，只需掌握常用语法，即可应付常见格式要求。
  • 支持多类型文章的导入导出，比如 Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。

平时写文章或者论文什么的，可以用 markdown 语法.让你全神贯注一心一意扑在写文章大业上，何乐而不为？

如果你问我有什么好的 Markdown 编辑器，那我绝对强烈推荐你用 Tlianjie ypora。

为什么呢，你可以点击这篇文章详细了解。接下来不多说，开始迎接主角。

常用 markdown 语法

1.标题

使用 # 可标记标题，其中#号的个数表示标题级别，最高级别是一级标题，最多支持 6 级标题。

2.字体

  1. 斜体文字可用 * * 表示。

  2. 粗体文字可用 ** **表示。

  3. 粗斜体文字可用 * * 表示

    其中*号也可用 _代替。

3.分割线

你可以在一行中用三个以上的星号、减号、底线来建立一个分隔线。

4.删除线

文字的两端加上两个波浪线 ~~ ,可为文字添加删除线。

5.下划线

文字的左端添加 ,右端添加 ,可为文字添加下划线

6.脚注

脚注是对文本的补充说明,它的格式为[^ ]

7.无序列表

无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容。

8.有序列表

有序列表使用数字并加上 . 号来表示.

9.链接
