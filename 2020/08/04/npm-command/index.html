npm 常见命令

一.npm&yarn 介绍

npm 是一个基于 Node.js 的包管理工具。
yarn 是 Facebook、Google、Exponent 和 Tilde 联合推出了一个新的 包管理工具,与 npm 相比有些优势。
npm 在安装 Node.js 时会被一并安装，yarn 则需要额外安装。

二.npm&cnpm 的关系

1.介绍

由于 npm 服务器在国外，所以在国内安装速度不是很理想，于是淘宝团队开发了一个 npmjs.org 的镜像，可以有效解决下载过于缓慢的问题。这是淘宝团队对它的介绍。

“这是一个完整 npmjs.org 镜像，你可以用此代替官方版本(只读)，同步频率目前为 10 分钟 一次以保证尽量与官方服务同步。”

2.安装

三.npm 常见命令

1.npm -v

查看 npm 版本信息。

2.npm install < Module Name > 本地安装

在本地目录的 node_modules 下安装模块。

3.npm install < Module Name > -g 全局安装

将模块安装在 node 所安装的目录。

4.npm list -g

查看全局安装的模块。

5.npm list < Module Name >

查看模块的版本信息。

6.npm uninstall < Module Name >

卸载模块。

7.npm update < Module Name >

更新模块。

8.npm search < Module Name >

搜索模块。

9.npm init

初始化创建模块

10.npm adduser

在某个资源库中注册账号。

11.npm pulish

推送模块到库。

四.nrm 包介绍

1.nrm 是一个源管理工具，利用它可以切换下载源。

2.安装

3.使用

1.nrm ls

列出所有的源。

2.nrm test

测试每个源的速度。

3.nrm use < registry >

切换 npm 源。

4.nrm add < registry > < url >

增加 npm 源。

5.nrm del < registry >

删除源。
